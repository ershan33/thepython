- 在 Python 中使用 `def`关键字定义函数。 把**用户自定义函数**的语句称为[[函数定义语句]]。 #定义
  id:: 6389debc-2fe8-4116-9c72-81ff2f64ba57
	- ```python
	  >>> def say_hello():
	  ...     print("hello")
	  ...
	  >>> say_hello()
	  hello
	  ```
-
- 函数定义并不会执行函数体；只有当函数被调用时才会执行此操作。 #初学者避坑指南
